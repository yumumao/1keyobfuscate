# 1keyobfuscate
一键混淆脚本
第 1 步：安装 Node.js
你需要先安装 Node.js，因为 javascript-obfuscator 是一个 Node.js 工具。

去官网下载安装：https://nodejs.org （选 LTS 版本，一路下一步即可）

安装完成后，打开终端验证：

bash
node -v
npm -v

能看到版本号就说明安装成功。

Windows 用户：打开 PowerShell 或 CMD
Mac 用户：打开 终端（Terminal）

第 2 步：安装混淆工具
bash
npm install -g javascript-obfuscator

等待安装完成，出现类似 added xx packages 就算成功。
